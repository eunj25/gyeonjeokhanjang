# 견적한장

> 디자인 프리랜서를 위한 견적서 자동화 도구

## 주요 기능

- 견적 항목 추가 / 자동 금액 계산
- 기준 단가표 직접 편집
- 디자인 등급 (심플 / 중급 / 고급) 선택
- 견적서 디자인 테마 3종 (A · B · C)
- 회사 로고 업로드
- 부가세 · 할인 · 유효기간 옵션
- 세금계산서 / 현금영수증 / 발급불가 선택
- 공급자 정보 브라우저 저장
- 라이트 / 다크 모드 토글
- PDF 출력 / 인쇄

## 로컬 실행

별도 빌드 없이 `index.html` 을 브라우저에서 바로 열면 됩니다.

```bash
open index.html
```
