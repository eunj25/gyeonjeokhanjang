견적한장 사용 가이드
기본 사용법
01
공급자 정보 입력
상호(회사명)·담당자 이름·연락처·이메일·계좌번호를 입력하세요.
로고 이미지는 클릭 또는 드래그로 업로드할 수 있습니다.
💾 저장 체크박스를 선택하면 다음 방문 시 자동으로 불러옵니다.
02
수신처 정보 입력
클라이언트명·담당자·연락처·프로젝트명을 입력하면 견적서에 즉시 반영됩니다.
03
기준 단가표 편집
항목명·단위·단가를 셀 클릭 후 직접 수정할 수 있습니다.
+ 항목 추가 버튼으로 새 항목을 넣고, × 버튼으로 삭제합니다.
04
디자인 등급 선택 후 항목 추가
심플(×1.0) / 중급(×2.0) / 고급(×3.0) 등급을 먼저 선택하면 단가가 자동 배율 적용됩니다.
항목을 선택하고 +를 누르면 견적 목록에 추가됩니다.
05
옵션 설정
부가세(VAT) 토글, 할인율(%), 유효기간(일), 세금계산서 발급 여부, 메모를 설정합니다.
06
견적서 디자인 선택
A(클래식) / B(인보이스) / C(헬로) 세 가지 테마 중 선택하면 오른쪽 미리보기가 즉시 전환됩니다.
07
출력 / PDF 저장
하단 ↓ 견적서 출력 버튼 또는 우상단 ⎙ 인쇄 버튼을 클릭합니다.
브라우저 인쇄 창에서 대상 → PDF로 저장을 선택하면 PDF 파일로 내보낼 수 있습니다.
